# Huffman-bmp

Небольшая утилита на Rust, которая:
- читает несжатое изображение `images.bmp`,
- считает частоты появления всех 256 возможных байтов,
- строит по ним коды Хаффмана
- и выводит получившиеся коды для каждого встречающегося байта.

## Запуск

cargo run
